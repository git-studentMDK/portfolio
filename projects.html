Student Management System

Student Management System is software which is helpful for school authorities to manage all day to day operations for a school. The Student management system is a real time java project. A student management software is a collection of computer instructions, specially designed to manage the day-to-day administrative tasks of schools.

Hospital Management System

The project Hospital Management system includes registration of patients, storing their details into the system. The software has the facility to give a unique id for every patient and stores the details of every patient and the staff automatically. It includes a search facility to know the current status of each appointment. Users can search availability of a doctor and the details of a patient using the id.

Image Caption Generator

This project aims to generate captions for images using deep learning techniques. It utilizes the Flickr30k dataset and a pre-trained VGG16 model for feature extraction, and a combination of CNN and LSTM models for caption generation.
